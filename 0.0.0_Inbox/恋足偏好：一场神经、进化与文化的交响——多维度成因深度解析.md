这是关于恋足癖最广为人知的神经科学假说，由著名神经科学家维莱亚努尔·拉马钱德兰（V.S. Ramachandran）在其对“幻肢”现象的研究中提出。

### <a name="总结从个人偏好到障碍的边界"></a>**总结：从个人偏好到“障碍”的边界**

2.  **导致功能损害**: 这种偏好严重影响了个体在社交、职业或其他重要功能领域的正常运作，例如，个体花费大量时间沉溺其中而无法工作，或引发了伤害他人的冲动且无法自控。
